IDR API

The Image Data Resource (IDR) provides a JSON-based API for accessing all datasets, thumbnails and metadata. This API is a combination of the standard OMERO.web API and newly developed search engine.

The code for the IPython notebook below is available on GitHub at IDR_API_example_script.ipynb as well as on the IDR virtual analysis environment and can also be run in public resource like mybinder.org.
