IDR and Analysis

Guides on how to access Image metadata and pixel data from IDR for analysis.

IDR access via the OMERO API

The IDR is based on the OMERO platform, which provides an open API for accessing image metadata and pixel data. Please see the README of the idr-notebooks repository for example notebooks that demonstrate how to access IDR data via the OMERO API.
